# CER Dashboard

The interactive dashboard shows:

- **CER Gauge** — Your context efficiency at a glance
- **Layer Distribution** — Token usage by kernel, skill, task, learning layers
- **Positional Attention Map** — Where critical instructions sit (start/middle/end)
- **Security Findings** — Active threats and suppressed false positives
- **What-If Simulator** — See projected CER when adding/removing content

Click action buttons directly in the dashboard to export, compare, or apply presets.
