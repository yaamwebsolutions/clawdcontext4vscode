# Support

## Where to Ask What

### Bug Reports

Use GitHub Issues and choose the bug report template.

Include:

- ClawdContext version
- VS Code version
- OS
- sample markdown files (sanitized)
- screenshots/logs

### Feature Requests

Use GitHub Issues and choose the feature request template.

Focus on:

- the problem
- why current behavior is insufficient
- expected UX/output

### Discussions (Recommended for Strategy / Research)

Use GitHub Discussions for:

- Markdown OS design patterns
- CER thresholds and heuristics
- AGENTS.md / CLAUDE.md / SKILL.md workflow architecture
- roadmap and ecosystem questions

### Security Reports

Follow `SECURITY.md` and use a private channel.

Do not post exploit details in public issues first.

## Commercial / Content / Website

The extension may live in a separate repository from `clawdcontext.com`. Use the extension repo for extension-specific support and the website repo for website issues.
